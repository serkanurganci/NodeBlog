# NodeJsBlog
A blog website which built with NodeJs and Handlebars.

## Dependencies

* body-parser: 1.19.0
* connect-mongo: 4.4.0
* express: 4.17.1
* express-fileupload: 1.2.1
* express-handlebars: 5.2.1
* express-session: 1.17.1
* method-override: 3.0.0
* moment: 2.29.1
* mongoose: 5.12.0
* nodemailer: 6.5.0
